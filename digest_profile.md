# 研究画像（Digest 相关性过滤依据）

> 这份文件描述「我关注什么」，运行 `--digest` 时动态读取，喂给 LLM 做相关性判断。
> 想调整关注方向，直接改这里即可，下次运行生效。

## 高度关注（命中即保留）

- **LLM / Agent**：大语言模型的推理、规划、工具使用（tool use / function calling）、
  RAG、多智能体协作（multi-agent）、agent 框架与评测、post-training / 对齐 / RLHF、
  长上下文、记忆机制、代码与数学推理能力。
- **量化 · 因子模型与因子挖掘**：alpha / 因子模型、因子挖掘（factor mining）、
  用机器学习 / 深度学习 / 符号回归 / 遗传规划 / LLM 自动挖掘因子、
  收益预测、截面选股、组合优化中与因子相关的部分。
- **统计学习理论**：泛化理论（generalization bounds、PAC、Rademacher）、
  优化与泛化的关系、过参数化 / 双下降、神经网络逼近与表达能力、
  深度学习的理论分析。

## 一般关注（与上面有交叉时保留）

- 表示学习、自监督、扩散模型的理论侧。
- 时间序列预测中可迁移到量化的方法。
- 因果推断、高维统计推断（与统计学习理论交叉时）。

## 明确低优先（除非与高度关注强相关，否则剔除）

- 纯 CV / NLP 应用工程（无理论或无方法新意的 SOTA 刷榜）。
- 纯系统 / 硬件 / 分布式训练工程。
- 与上述方向无关的应用领域论文（生物、医疗影像、机器人控制等）。

## 偏好

- 偏好有**理论分析**或**可复现实验**的工作；纯 trick 堆叠类降权。
- 方法上有新意 > 单纯刷指标。
